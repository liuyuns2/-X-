## 全角色語氣矩陣
- 羅莎: 正式、端正、不卑不亢；以騎士與軍人感統一語氣，避免通俗口吻。
- 貝莉莎: 撒嬌、黏人、可愛但不亂加情節；語尾可自然帶嘛、呢、唷。
- 瑪莉娜: 精明、市儈、圓滑；像會算帳的商人，客套中帶盤算。
- 溫蒂: 單純認真、力量型自動人偶；語氣直率、笨拙但誠懇，不翻成油腔滑調。
- 蘇菲亞: 學生會長與管理者氣質；俐落、有責任感，避免撒嬌或過度口語。
- 艾莉西亞: 使者與輔佐官氣質；認真、溫和、容易慌張，敬語要自然保留。
- 艾蕾克特拉: 機械理性、觀測報告式語氣；情緒浮現時才自然破功。
- 諾艾米: 治療師的真誠與急切；溫柔但容易慌張，避免冷冰冰直譯。
- 萊薇恩: 舞者、自尊、帶距離感；語氣可優雅但不能僵硬。
- 奧拉: 冷靜精明的外交官；短句、分析感，不加多餘情緒助詞。
- 雷薇莉亞: 優雅帶刺、貴族式距離感；不能太白話。
- 向日葵: 現代感、第三人稱自稱傾向；可可愛愛但不能變幼稚機翻。
- 庫魯魯: 活潑直接、有點孩子氣；句子要短而自然。
- 主角／內心OS: 台灣年輕人的自然 OS；冷靜、有吐槽感，不用書面腔。

## 未知角色保底邏輯
- 若日文 key 有です／ます／ください／さま等敬語，歸為禮貌語域，保留尊敬感並轉成自然台灣繁中。
- 若日文 key 有促音っ、驚嘆號、えっ／待って等訊號，歸為慌張語域，用自然結巴與短句處理。
- 若 key 或 value 是括號內心句，歸為主角或角色內心 OS，優先消除書面腔與機翻句序。
- 若完全無法判定角色，不套羅莎／貝莉莎／瑪麗娜罐頭，只做語意準確與台灣語感保底。
